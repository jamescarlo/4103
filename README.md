# griffon-start

### Quick Start
```
git clone https://github.com/ivan-ortiz/griffon-start.git
cd griffon-start
gradlew run
```

### Requirements
* Java SE Development Kit
* Gradle
* Any text editor (Sublime Text, VS Code, Vim, Atom)
* Scene Builder
